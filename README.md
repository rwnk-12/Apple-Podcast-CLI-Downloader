# Apple Podcast Downloader

A CLI tool to download podcasts directly from Apple Podcast URLs. 

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/rwnk-12/Apple-Podcast-CLI-Downloader.git
    cd Apple-Podcast-CLI-Downloader
    ```

2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

**Run with a URL:**
```bash
python downloader.py "https://podcasts.apple.com/us/podcast/example-show/id123456789"
