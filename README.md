# Apple Podcast Downloader

A CLI tool to download podcasts directly from Apple Podcast URLs. [MP3 128K]

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/rwnk-12/Apple-Podcast-CLI-Downloader.git
    cd Apple-Podcast-CLI-Downloader
    ```

2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

**Run with a URL:**
```bash
python downloader.py "https://podcasts.apple.com/us/podcast/life-lessons-from-history/id1856559246"
